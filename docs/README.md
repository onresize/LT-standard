# LT-standard 前端项目开发规范

## 前言

此规范包含了四个部分：JavaScript开发规范、CSS / Scss 开发规范、Vue 项目开发规范及项目通用规范

借鉴了行业内较为流行的前端规范及官方的style guide，并基于此进行改动，更适合各个中小型团队日常的项目开发工作
